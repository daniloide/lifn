Estructura del sistema¶

El Inventario Nacional Forestal esta compuesto por dos aplicaciones:

  1. Una aplicación web que integra la información recogida en campo y permite obtener resultados acerca del estado de los recursos forestales del país (denominada IFN).

  2. Una aplicación de escritorio (denominada LInFor) capaz de ejecutarse sobre los sistemas operativos Windows, GNU/Linux y Mac que contiene:
    • un pequeño servidor web;
    • una aplicación web para el ingreso de datos de campo (denominada LIFN).

Aplicación IFN¶

Esta aplicación integra la información histórica de datos forestales del país.

Permite realizar consultas predefinidas por el usuario del tipo:

  • temporal;
  • espacial;
  • estadísticas.

Posee salidas:

  • gráficas: mapas y/o gráficos;
  • tabulares que peuden ser exportadas a planilals de cálculo.

Aplicación LInFor¶

  1. Una aplicación de escritorio web que integra la información recogida en campo y permite obtener resultados acerca del estado de los recursos forestales del país.
  2. Una aplicación de escritorio capaz de ejecutarse sobre los sistemas operativos Windows, GNU/Linux y Mac que contiene un servidor web (LINFor);
  3. Una aplicación web (depende de la anterior) que permite completar los datos recabados en campo (LIFN);
