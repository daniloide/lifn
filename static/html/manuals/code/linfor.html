Documentacion del código fuente de LInFor¶

LInFor es una aplicación de escritorio multiplataforma (Windows, GNU/Linux, Mac) desarrollada con PySIde, que inicializa un servidor web (Rocket) que permite ejecutar aplicaciones construidas con web2py.

El núcleo de la aplicación es el módulo igs. Su documentación se muestra a continuación.
